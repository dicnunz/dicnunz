# Nicholas Dunzelman

I use AI heavily to make experiments, weird tools, and fast product-like prototypes.

Start here: [dicnunz.github.io](https://dicnunz.github.io/)

Most of the code here is AI-assisted. The throughline is not deep expertise in every domain these repos touch. It is idea selection, iteration speed, packaging, and pushing experiments far enough to become believable demos.

## Best public work

- [mentor-worker-benchmark](https://github.com/dicnunz/mentor-worker-benchmark): AI-assisted benchmark experiment around mentor-worker LLM collaboration. [Live leaderboard](https://dicnunz.github.io/mentor-worker-benchmark/)
- [taskproof](https://github.com/dicnunz/taskproof): AI-assisted browser proof prototype for turning task specs into inspectable evidence bundles.
- [boundary-atlas](https://github.com/dicnunz/boundary-atlas): AI-assisted architecture-analysis experiment for TS/JS repos.
- [counterexample-studio](https://github.com/dicnunz/counterexample-studio): AI-assisted testing experiment around reproducible counterexamples and shrink traces.
- [llm-eval-harness](https://github.com/dicnunz/llm-eval-harness): AI-assisted CLI experiment for local-LLM evals over OpenAI-compatible APIs.
- [pixelmelt](https://github.com/dicnunz/pixelmelt): AI-assisted creative prototype for image-to-material simulation and clip export.
